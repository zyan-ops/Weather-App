# 🌦️ Weather App (React + Chart.js)

A modern, glassmorphism-style **Weather Application** built using **React**, **Chart.js**, and **OpenWeatherMap API**.  
It provides real-time weather data, hourly forecasts, and a clean UI with smooth charts.

---

## 🚀 Features

✨ Auto Location Detection  
- Uses browser geolocation API 📍  
- Automatically fetches weather of current location  

🔍 City Search  
- Search weather by city name  
- Instant updates with validation  

📊 Temperature Chart  
- Smooth line chart using Chart.js  
- Displays hourly temperature trend  

📅 5-Day Forecast  
- Clean daily forecast cards  
- Weather icons + temperature  

💧 Weather Details  
- Temperature (°C)  
- Humidity (%)  
- Wind Speed (km/h)  
- Weather condition  

🎨 Modern UI  
- Glassmorphism design  
- Gradient glow effects  
- Fully responsive layout
